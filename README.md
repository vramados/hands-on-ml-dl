# hands-on-ml-dl

Five projects that explain basic Machine Learning (ML) and Deep Learning (DL) foundations:

1. **Linear Regression from Scratch (ML)**
   - Build linear regression with NumPy to learn supervised learning, loss functions, and gradient descent.
2. **Binary Classification with Logistic Regression (ML)**
   - Train a classifier for spam or churn prediction to understand classification metrics and decision boundaries.
3. **Decision Trees and Random Forests (ML)**
   - Compare a single tree with an ensemble model to explore overfitting, feature importance, and generalization.
4. **Image Classification with a Simple CNN (DL)**
   - Use a convolutional neural network on MNIST/CIFAR-10 to learn tensors, backpropagation, and training loops.
5. **Text Sentiment Analysis with RNN/LSTM (DL)**
   - Build a sequence model for sentiment classification to understand embeddings, sequence modeling, and recurrent layers.
